\section*{\fontsize{16pt}{15pt}\selectfont SENTIMENT ANALYSIS CLASSIFICATION WITH YOUTUBE API: \\DATA COLLECTION, NORMALIZATION AND LABELING \\OF MOVIE COMMENTS \\ABSTRACT}

\begin{itemize}
    \item[-] \textbf{Research Problem:}
    
    This project develops an automated sentiment analysis system for YouTube comments using YouTube Data API v3 and TextBlob library. The system implements a 6-step pipeline: data collection, text normalization, sentiment labeling, model experimentation, application development, and deep result analysis. Data is collected from "Avengers: Endgame" (2019) video with 100,000 comments.
    
    The report consists of 5 chapters:
        \begin{itemize}
        \item[•] Chapter 1 - Introduction: Overview of sentiment analysis, objectives and scope
        \item[•] Chapter 2 - Data Collection and Normalization: YouTube API, 8-step cleaning process
        \item[•] Chapter 3 - Labeling: TextBlob methodology, 3 confidence levels
        \item[•] Chapter 4 - Experimentation and Application: Feature relationship analysis, demo development
        \item[•] Chapter 5 - Deep Analysis and Conclusion: Insights, evaluation and future directions
        \end{itemize}
        
    \item[-] \textbf{Approach:}
        \begin{itemize}
        \item[•] Use YouTube Data API v3 with 4 backup API keys
        \item[•] Apply 8-step text normalization (remove URLs, mentions, hashtags, special characters)
        \item[•] Label sentiment with TextBlob (Positive/Negative/Neutral)
        \item[•] Analyze correlation between length, likes and sentiment score
        \item[•] Visualize data with 6 charts (bar, pie, scatter, box plot)
        \item[•] Build demo application for new text analysis
        \end{itemize}
        
    \item[-] \textbf{Solution:}
    
    Using Python with Pandas, NumPy, TextBlob, Matplotlib and Seaborn libraries. Collect 100,000 comments from YouTube API, then clean using regex and string processing. Label sentiment based on TextBlob's polarity score with 3 confidence levels (High/Medium/Low). Analyze feature relationships using correlation analysis, distribution analysis and regression analysis. Build console application allowing users to input text and receive instant analysis results.
    
    \item[-] \textbf{Key Results:}
        \begin{itemize}
        \item[•] Successfully collected 100,000 comments, 92,521 valid after normalization
        \item[•] Sentiment distribution: Neutral 59.5\%, Positive 28.6\%, Negative 11.9\%
        \item[•] Positive comments have highest average length (85.3 characters)
        \item[•] Positive comments receive more likes (average 12.8 likes)
        \item[•] Weak correlation between length and sentiment score (r = 0.15)
        \item[•] Successfully built demo application with user-friendly console interface
        \item[•] Created 6 detailed analysis charts about distribution and data relationships
        \item[•] System processes and analyzes text in real-time (< 1 second)
        \end{itemize}
\end{itemize}
